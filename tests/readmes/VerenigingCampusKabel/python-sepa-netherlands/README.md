# Python SEPA Netherlands library

Python library which handles communication with Dutch banks for processing SEPA eMandates.

## Technical support
- [ ] Business to Core (B2C)
    - [ ] Directory protocol
    - [ ] Transaction protocol
    - [ ] Status procotol
- [ ] Business to Business (B2B)
    - [ ] Directory protocol
    - [ ] Transaction protocol
    - [ ] Status procotol
