Rapidly create services.

*[Framework]*

`Orile-Service <https://github.com/zhouxiaoxiang/oriole>`_.

* Run a service.

  .. code:: python

  oriole run <service>

* Run unit tests.

  .. code:: python

  oriole test

* Create documents.

  .. code:: python

  oriole doc
