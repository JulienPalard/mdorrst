Py Zip Code API
===============================

version number: 1.0.0

author: Mohamed Ben Makhlouf

python version: > 2.7

Documentation
--------

link : http://pyzipcodeapi.readthedocs.io/en/v1.0.0/

Installation / Usage
--------------------

To install use pip:

    $ pip install pyzipcodeapi


Or clone the repo:

    $ git clone https://github.com/medbenmakhlouf/pyzipcodeapi.git
    $ python setup.py install
